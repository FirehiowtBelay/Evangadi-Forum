🧠 AI-Powered Forum System

A modern full-stack forum platform designed for seamless communication, discussion, and knowledge sharing. This system allows users to register, post questions, 
provide answers, and interact in a structured community environment.

🚀 Features
👤 User authentication (Register / Login / Logout)
📝 Create, edit, and delete posts (questions)
💬 Answer and comment on discussions
🔍 Search functionality for posts
📊 User profiles with activity tracking
🔐 Protected routes and role-based access
⚡ RESTful API backend
🎨 Responsive and user-friendly UI

Frontend

HTML5, CSS3, JavaScript (ES6+)
Axios / Fetch API
Bootstrap / Tailwind CSS (if used)
Backend
Node.js
Express.js
MongoDB / MySQL (depending on your setup)
